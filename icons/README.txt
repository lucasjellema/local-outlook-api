Icon Requirements for Chrome Extension

You'll need to create the following icon files in this directory:

1. icon16.png - 16x16 pixels
2. icon48.png - 48x48 pixels
3. icon128.png - 128x128 pixels

These icons should be in PNG format with transparency.

You can create these icons using any image editor like Photoshop, GIMP, or online tools like Canva.

For a quick solution, you can download free icons from sites like:
- https://www.flaticon.com/
- https://icons8.com/
- https://material.io/resources/icons/

Recommended icon style:
- Simple, clean design
- Blue color scheme to match Outlook's branding
- Email or messaging related imagery
